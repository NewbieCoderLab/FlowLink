# FlowLink

FlowLink 是一个面向 macOS 和 Windows 的跨设备协同控制应用，目标是在同一局域网内让多台电脑像一组连续屏幕一样协同工作，实现鼠标控制权在设备之间的平滑切换。

项目采用 `Tauri + Rust + React + TypeScript` 构建。Rust 负责低延迟输入、网络协议、设备状态和本地存储等核心能力，React 前端负责设备列表、权限状态、配对、布局配置和诊断面板。

> 当前仓库处于 MVP 早期开发阶段，已完成基础工程和核心模块骨架，尚未形成可用于日常工作的正式版本。

## 功能范围

V1 目标功能：

- 局域网内自动发现 macOS / Windows 设备
- 通过配对确认建立可信设备关系
- 配置远端设备相对本机的上下左右布局
- 鼠标移动到指定屏幕边缘后切换控制设备
- 转发鼠标移动、点击、释放和滚轮事件
- 维护连接状态、心跳检测和断线重连

V1 不包含：

- 账号系统
- 公网中继或 NAT 穿透
- 文件传输
- 剪贴板同步
- 键盘共享
- 三台及以上设备布局

## 项目结构

- `src/`：React + TypeScript 前端界面
- `src-tauri/`：Rust 核心逻辑与 Tauri 桌面端
- `docs/`：产品、架构、协议、存储、路线图等设计文档

## 技术栈

- 桌面框架：Tauri 2
- 核心逻辑：Rust
- 前端界面：React + TypeScript
- 构建工具：Vite
- 本地存储：JSON 文件
- 实时通信目标：TCP 长连接 + 长度前缀二进制协议
- 设备发现目标：mDNS / Zeroconf，UDP Broadcast 作为兜底

## 本地开发

安装前端依赖：

```bash
npm install
```

启动桌面应用开发模式：

```bash
npm run dev
```

仅启动前端开发服务：

```bash
npm run web:dev
```

构建前端资源：

```bash
npm run build
```

运行 Tauri 应用需要本机具备 Rust 和 Tauri 相关开发环境。具体安装方式请参考 Tauri 官方文档。
