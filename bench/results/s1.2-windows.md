Date: 2026-06-12

- Low-level mouse hook capture.
- SendInput move, click, wheel, and horizontal wheel injection.
- Self-injection filtering through `dwExtraInfo == FLOW_SIGNATURE`.
- DPI-aware screen topology and virtual desktop coordinate normalization.

Current result:

- Automated checks pass on the current Windows development environment.
- The spike prints 1000 hook events.
- Move injection plan sends 100 visible relative moves.
- Click injection plan sends left, right, and middle down/up at the current cursor position.
- Wheel injection plan covers vertical up/down and horizontal right/left.
- Self-filter check drains the local queue, injects tagged moves, then verifies no injected events reach the capture queue.
- `--warp-corners` is available for manual multi-monitor absolute-position checks.

Known limitation:

- Normal-integrity FlowLink cannot reliably inject into elevated administrator windows because of Windows UIPI. The Permissions tab now exposes the current process integrity level so testers can identify this case.
